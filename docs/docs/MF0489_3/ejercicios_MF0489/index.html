Instalar Servidor OpenSSH en Linux​

Actualizar sistema​

Como siempre antes de realizar cualquier acción es aconsejable tener el sistema actualizado y las fuentes de los repositorios con las últimas versiones de los paquetes disponibles. Por ello realizar

apt update
apt upgrade
apt dist-upgrade

Comprobar que si SSH esta instalado​

Primero se debe de verificar si SSH ya ha sido instalado en el sistema y si esta habilitado. Esta verificación se puede realizar comprobando los servicios en ejecución.

service --status-all

Para instalar el servidor podemos usar el siguiente comando

sudo apt -y install openssh-server

Para eliminar en caso necesario la configuración de openssh-server, los datos y todas sus dependencias, podemos utilizar el siguiente comando:

sudo apt -y autoremove --purge openssh-server

Una vez instalado, hay que configurar el servidor ssh para que se ejecute al arrancar el sistema.

systemctl enable ssh

Para inicializar el servidor debemos de introducir el comando:

service ssh start

verificando a continuación si se ha inicializado correctamente.

service ssh status
